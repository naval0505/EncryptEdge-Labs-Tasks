
# Phase E – Hosted Labs Integration

## Objective

The objective of this phase was to reinforce the concepts of network reconnaissance, offensive security methodology, and laboratory safety through guided hands-on exercises on the TryHackMe platform.

These hosted labs complemented the virtual lab setup by providing practical experience with reconnaissance techniques, network scanning, and the ethical use of offensive security tools in controlled environments.

# Lab 1 – Offensive Security Intro

## Objective

The **Offensive Security Intro** room provides an introduction to penetration testing concepts, offensive security methodologies, and ethical hacking principles.

The lab focuses on understanding how attackers think, the stages of a penetration test, and the importance of conducting security assessments within authorized environments.

### Topics Covered

- Offensive Security Methodology
- Ethical Hacking Principles
- Penetration Testing Process
- Responsible Disclosure
- Legal Scope of Testing

### Skills Gained

- Understanding Red Team objectives
- Ethical hacking fundamentals
- Attack lifecycle awareness
- Responsible security testing

---

# Lab 2 – Nmap

## Objective

The **Nmap** room introduces one of the most widely used network reconnaissance tools in cybersecurity.

Participants learn how to identify live hosts, discover open ports, detect running services, and understand how reconnaissance supports later stages of a penetration test.

### Topics Covered

- Host Discovery
- Port Scanning
- Service Enumeration
- Version Detection
- Network Reconnaissance

### Skills Gained

- Performing basic Nmap scans
- Identifying exposed services
- Understanding network exposure
- Reconnaissance best practices

# Practical Observations

The hosted labs demonstrated that successful penetration testing begins with accurate reconnaissance and disciplined methodology rather than immediately attempting exploitation.

Important observations included:

- Information gathering is a critical first step.
- Open ports reveal potential attack surfaces.
- Service enumeration helps identify vulnerabilities.
- Reconnaissance should always remain within authorized scope.
- Proper documentation improves repeatability and reporting.

# Lessons Learned

Completing the hosted labs reinforced the importance of preparation before exploitation. Effective penetration testing depends on gathering accurate information, understanding the target environment, and maintaining strict adherence to ethical and legal boundaries.

The labs also demonstrated how reconnaissance techniques integrate with a properly isolated Red Team laboratory, allowing offensive security testing to be performed safely and professionally.

---

# Phase Summary

This phase successfully reinforced the theoretical concepts introduced throughout the task by applying them within guided TryHackMe environments. The hosted labs improved practical understanding of reconnaissance, offensive security methodology, and operational discipline while complementing the isolated laboratory environment developed in previous phases.

The knowledge gained from these exercises provides a strong foundation for future penetration testing engagements and advanced Red Team operations.
